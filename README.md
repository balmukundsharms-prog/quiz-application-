A lightweight, interactive console-based [or GUI-based, depending on your build] quiz application built using Java. This project tests users across various categories, tracks scores in real-time, and provides immediate feedback upon completion.

Features
Dynamic Question Loading: Supports multiple-choice questions (MCQs) with varied difficulty levels.

Real-Time Scoring: Automatically calculates and tracks the user's score as they progress.

Immediate Feedback: Shows correct answers at the end of the quiz [or right after each question].

Object-Oriented Design: Built using clean OOP principles (Encapsulation, Inheritance, and Polymorphism) making it modular and easy to extend.

Architecture Overview
The application follows a structured approach to separate data, logic, and user interaction:

Question Class: Encapsulates the question text, options, and the correct answer index.

Quiz Class: Manages the collection of questions, tracks the current state, and handles score evaluation.

Main / App Class: Serves as the entry point, managing the user interface loop and input validation.
